## ✨ Key Features

- **🎨 Premium UI/UX Design**: High-end visual aesthetic utilizing glassmorphism, dynamic glowing gradients, and perfectly balanced typography.
- **🌗 Dark/Light Mode**: Full theme support with seamless transitions, deep navy/slate backgrounds, and vibrant neon accents (Cyan/Blue/Purple).
- **🚀 Advanced Animations**: Complex scroll-triggered animations, interactive hover states, typing effects, and floating elements powered by `framer-motion`.
- **📱 Fully Responsive**: A mathematically precise layout grid that scales flawlessly from ultra-wide 4K desktop monitors down to mobile devices.
- **⚡ Performance Optimized**: Built on Next.js App Router for incredibly fast page loads, optimized image rendering, and SEO best practices.
- **🔌 Live GitHub Integration**: Automatically fetches and renders live repository data directly from GitHub using the GitHub API.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (React)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)

- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/) (Recommended)

## 🏗️ Local Development

Follow these steps to run the portfolio on your local machine:

**1. Clone the repository**
```bash
git clone https://github.com/mt-abdulla-it/portfolio.git
cd portfolio
```

**2. Install dependencies**
```bash
npm install
# or
yarn install
```

**3. Run the development server**
```bash
npm run dev
# or
yarn dev
```

**4. View the application**
Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 📐 Design Architecture

The layout follows strict, professional UI guidelines:
- **Global Container**: Hard-capped `1200px` max-width to ensure perfect readability on ultra-wide monitors.
- **Left-Aligned Axis**: All typography and section headers strictly align to the left grid axis, providing a professional SaaS reading rhythm.
- **Equal-Height Cards**: Grid elements utilize `h-full` to maintain structural integrity regardless of content length.
